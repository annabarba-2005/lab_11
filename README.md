# Лабораторная работа по web-дизайну (группа 301, студентка 3 курса)

## Описание проекта
Адаптивный одностраничный сайт-портфолио о путешествиях по России, созданный по макету из Figma.  
Сайт корректно отображается на экранах различных устройств — от мобильных (320px) до широких экранов (1280px).

## Используемые технологии
- HTML / CSS  
- JavaScript  
- Grid Layout  
- Flexbox  
- Медиазапросы  
- Подключение шрифтов  
- Методология БЭМ (Nested)

## Инструкция по запуску
Откройте файл `index.html` в любом современном браузере.
